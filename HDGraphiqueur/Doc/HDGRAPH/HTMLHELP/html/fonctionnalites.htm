Fonctionnalités


  • Affichage des sous-répertoires d'un répertoire donné, sous forme d'anneaux (ou camemberts imbriqués, selon le point de vue), proportionnellement à l'espace du répertoire parent.
  • Affichage de n niveaux de sous-répertoires sur le même graphique. Nombre de niveaux paramétrable avec application immédiate.
  • Intégration à l'explorateur windows, pour afficher le graphique à l'aide d'un simple clic droit sur un répertoire.
  • Possibilité de sauvegarder le graphique pour chargement ultérieur, indépendament du nouvel état de l'arborescence des fichiers.
  • Possibilité d'exporter le graphique sous forme d'image (BMP).
  • Manipulation interactive du graphique:
    • Possiblité de centrer la vue sur un répertoire donné via double-clic ou clic droit.
    • Possiblité de centrer la vue sur le répertoire parent du répertoire central via double-clic ou clic droit.
    • Possiblité d'ouvrir l'explorateur windows sur un répertoire donné directement depuis un clic droit sur le graphique.
  • Rafraichissement automatique du graphe lors du passage d'un répertoire à l'autre (si le nouveau répertoire n'avait pas été précédement chargé).
  • Gestion d'un historique des répertoires consultés.
  • Interface graphique multilingue. Changement de langue avec application immédiate.
